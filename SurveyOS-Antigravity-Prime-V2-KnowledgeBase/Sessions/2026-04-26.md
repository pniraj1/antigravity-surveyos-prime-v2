## Key Decisions

> [!tip] Decisions Made
> 1. **No sandbox** ‚Äî live fetch + DocModeToggle covers the use case cleanly
> 2. **`AIControls.tsx` as shared lib** ‚Äî prevents DocumentsTab and AssessmentTab diverging
> 3. **`aiDocMode` in profile** ‚Äî persists across sessions, user sets it once
> 4. **No timeout on extraction** ‚Äî long PDFs need patience, not hard failure
> 5. **3 Gemini models only** ‚Äî live fetch already surfaces account-specific models; static list is noise-free fallback
> 6. **NVIDIA NIM for vault logs** ‚Äî free model, zero Claude tokens burned on logging
